A desktop with limitless bounds. Picture GIMP but for windows. WIMP, if you will.

A Wayland compositor based on wlroots.

This software is a work in progress.

Building
--------

Dependencies:
 - wayland
 - wayland-protocols
 - wlroots
 - cairo

.. code-block:: sh

   make
   sudo make install
